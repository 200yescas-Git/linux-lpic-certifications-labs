# Linux Professional Institute Certification (LPIC)

Las certificaciones LPIC (Linux Professional Institute Certification) son credenciales reconocidas internacionalmente que validan conocimientos y habilidades en administración de sistemas Linux. Desarrolladas por el Linux Professional Institute (LPI), estas certificaciones abarcan desde fundamentos de Linux hasta la administración avanzada de infraestructuras empresariales.

La ruta de certificación LPIC está diseñada para proporcionar una progresión estructurada en el aprendizaje de Linux y tecnologías de código abierto, permitiendo a los profesionales desarrollar competencias aplicables en entornos reales de TI.

## Ruta de Certificación

### Linux Essentials
Introducción a los conceptos fundamentales de Linux, software libre, línea de comandos, administración básica del sistema y principios del código abierto.

### LPIC-1: Linux Administrator
Valida habilidades esenciales para administrar sistemas Linux, incluyendo instalación, gestión de paquetes, sistemas de archivos, usuarios, permisos, redes básicas y tareas administrativas.

### LPIC-2: Linux Engineer
Se enfoca en la administración avanzada de sistemas Linux, redes empresariales, servicios de red, almacenamiento, seguridad y resolución de problemas en entornos de producción.

### LPIC-3: Enterprise Professional
Nivel avanzado con especializaciones orientadas a entornos empresariales, incluyendo seguridad, virtualización, alta disponibilidad, automatización e integración de infraestructuras complejas.

## Laboratorios

Los laboratorios incluidos en este repositorio tienen como objetivo reforzar los conocimientos teóricos mediante ejercicios prácticos realizados en entornos Linux.

Las prácticas abarcan tareas relacionadas con:

- Navegación y administración del sistema de archivos.
- Gestión de usuarios y grupos.
- Configuración de permisos y propiedad de archivos.
- Administración de procesos y servicios.
- Gestión de almacenamiento y particiones.
- Configuración básica de redes.
- Automatización mediante scripts.
- Seguridad y administración de sistemas Linux.

Cada laboratorio incluye documentación, comandos utilizados, resultados obtenidos y evidencia cuando sea necesario.
