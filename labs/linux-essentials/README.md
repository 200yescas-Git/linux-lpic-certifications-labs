# Linux Essentials – Labs

Esta sección contiene los laboratorios prácticos correspondientes a la certificación Linux Essentials (010-160) del Linux Professional Institute (LPI).

## Objetivo

El objetivo es reforzar los conocimientos teóricos mediante la práctica en un entorno Linux real, aplicando comandos, herramientas y conceptos fundamentales de administración básica de sistemas.

---

## Entorno de laboratorio

Todos los laboratorios se realizan en un entorno controlado para asegurar consistencia y repetibilidad:

- VMware Workstation
- Debian GNU/Linux
- Terminal Bash
- Usuario con privilegios sudo
