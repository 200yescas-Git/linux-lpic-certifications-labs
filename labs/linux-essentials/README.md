# Linux Essentials – Labs

Esta sección contiene los laboratorios prácticos correspondientes a la certificación Linux Essentials (010-160) del Linux Professional Institute (LPI).

## Objetivo

El objetivo es reforzar los conocimientos teóricos mediante la práctica en un entorno Linux real, aplicando comandos, herramientas y conceptos fundamentales de administración básica de sistemas.

---

## Entorno de Laboratorio

Para optimizar recursos y mantener consistencia entre las distintas certificaciones de mi ruta de aprendizaje, los laboratorios de este repositorio se realizan utilizando la misma máquina virtual Debian empleada en mis estudios de Linux Essentials, CompTIA Linux+ y LPIC.

Todos los laboratorios se realizan en un entorno controlado para asegurar consistencia y repetibilidad.

## Plataforma utilizada

- VMware Workstation
- Debian GNU/Linux
- Terminal Bash
- Usuario con privilegios sudo

## Áreas de práctica

- Introducción a Linux y uso de la terminal  
- Sistema de archivos y gestión de archivos  
- Comandos básicos, pipes y redirecciones  
- Usuarios, grupos y permisos  
- Procesos y administración del sistema  
- Redes básicas en Linux
