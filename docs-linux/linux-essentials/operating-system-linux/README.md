# Características del Sistema Operativo Linux 

## Overview

En esta sección vamos a conocer la línea de comandos,ubicar el intérprete de comandos para poder iniciar nuestras primeras prácticas.

---

## Learning Objectives

- La línea de comandos
- Acceso al emulador de la terminal
- Conocer el prompt de Bash
- Construcción de una orden
- Comandos principales a ejecutar
- Obtener ayuda de las páginas MAN
- Navegar por los directorios del sistema
- Tipos de comandos
- Globbing

## Summary

Aprender comandos y ejecutarlos para llevar una buena administración con base a la práctica, conocer el entorno de la línea de comandos es el primer paso en el mundo linux.
