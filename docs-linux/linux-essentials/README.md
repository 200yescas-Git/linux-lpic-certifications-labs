# 📘 Linux Essentials – Documentation

Esta sección contiene la documentación estructurada correspondiente a la certificación Linux Essentials (010-160) del Linux Professional Institute (LPI).

El objetivo es construir una base sólida en el uso de sistemas Linux, la línea de comandos y conceptos fundamentales de administración de sistemas.

---

## 📚 Dominios de conocimiento

### 1. The Linux Community and a Career in Open Source
- Historia de Linux y Unix
- Distribuciones Linux
- Software libre y open source
- Licencias (GPL, MIT, etc.)
- Comunidad Linux
- Carreras en IT

### 2. Finding Your Way on a Linux System
- Estructura del sistema de archivos (FHS)
- Rutas absolutas y relativas
- Navegación en terminal
- Gestión de archivos y directorios
- Uso de manuales (man pages)

### 3. The Power of the Command Line
- Shell y Bash
- Ejecución de comandos
- Pipes y redirecciones
- Búsqueda de archivos y texto
- Compresión y archivos
- Introducción a scripting

### 4. The Linux Operating System
- Arquitectura del sistema
- Procesos
- Memoria básica
- Dispositivos y almacenamiento
- Redes básicas

### 5. Security and File Permissions
- Usuarios y grupos
- Permisos (rwx)
- chmod, chown, umask
- Seguridad básica
- Buenas prácticas

## 🧪 Laboratorios

Los laboratorios tienen como objetivo reforzar los conceptos teóricos mediante práctica directa en un entorno Linux (Debian en VMware Workstation).

### Ejemplos de laboratorios:

- Navegación en el sistema de archivos
- Gestión de archivos y directorios
- Uso de comandos básicos de Linux
- Gestión de usuarios y grupos
- Configuración de permisos
- Uso de redirecciones y pipes
- Búsqueda de archivos y texto
- Compresión y descompresión de archivos

Cada laboratorio incluye:
- Objetivo
- Comandos utilizados
- Resultados
- Evidencia en terminal (cuando aplique)

---

## 📌 Resumen

Linux Essentials proporciona los fundamentos esenciales para el uso del sistema operativo Linux y la administración básica de sistemas.

Los conocimientos adquiridos en esta certificación sirven como base para niveles superiores como CompTIA Linux+, LPIC-1 y certificaciones enfocadas en administración de sistemas, ciberseguridad y DevOps.
